## Live2dRender

[English Document](https://www.npmjs.com/package/live2d-render)

适用于最新版本 Live2D 模型文件的 Javascript 渲染器。


:::info
- 此项目不适用于 IE 浏览器。
- 这玩意儿是基于 `CubismSdkForWeb` 开发的，请不要商用！
- 使用前请确保你的 Live2d 模型的文件都是英文，如果是中文可能会出问题。
:::
